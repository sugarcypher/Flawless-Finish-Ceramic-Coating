Security

How we protect your data (demo deployment).

  • All pages served over HTTPS by the hosting provider.
  • Security headers via helmet: X-Content-Type-Options, X-Frame-Options, Referrer-Policy, and more.
  • Rate limiting on API routes to reduce abuse.
  • No real payment data is processed in this demo. Deposits are simulated only.
  • Minimal data collection: name, phone, and selected slot for bookings.
  • On production, use a managed database (PostgreSQL) instead of local JSON files.
